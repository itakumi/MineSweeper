Mine.zipを解凍してMineフォルダの中にあるMine.exeを実行することでプレイできます。
Mineフォルダ内の構造はそのままにしてください。
操作手順は以下です
1.ユーザー名入力
ユーザー名を入力してください。一つの名前では一つのアカウントしか作ることができません。
2.プレイ画面
ユーザー名を入力すると初級のゲームが開始します。タイトルの部分にはレベルとユーザー名が表示されています。
ゲームの難易度を変えたい場合は「難易度選択」バーから難易度を選択、あるいはカスタムでお好みの値でプレイできます。
カスタムは縦横それぞれ50マス以下の整数、地雷の数は1個以上マスの数未満です。
マス目の大きさも「マス目の大きさ」バーによって設定することができます。(マス目の大きさは変更するとその次のゲームより大きさが変更されます)

マインスイーパーの盤面の上部分のエリアは左から残りの地雷数、ニューゲームボタン、経過時間、現在の取得経験値となっています。
ゲームの点数について
以下のようにすればするほど点数は高くなります。
・ミスをできるだけしない
・経過時間が少ない
・地雷の数が多い

閉じるときはウィンドウの×ボタンで閉じてください
